1. 	Install Python!
1.1 Go to the Link: https://www.python.org/downloads/
1.2 Download the latest Python version (3.13.5 atm, but everything from python 2.0 works!)
1.3 Open the Launcher
1.4 Add a check to python.exe to PATH
1.5 Open the customize installation
1.6 Add a check to pip, so you get the pip install tool
1.7 Choose location
1.8 Press Install

2. 	Install PyGame!
2.1	open the CMD 
2.2 check python with python --version
2.3 use following command to install PyGame: python3 -m pip install -U pygame --user
2.4 Check if everything is alright
2.5 open IDLE Shell
2.6 import pygame
2.7 If you get a message with a welcome, then u succesfully installed PyGame, congrats!
2.8 Open the folder in you IDE 
2.9 And run the main file -> Teti.py  

Enjoy the game!
